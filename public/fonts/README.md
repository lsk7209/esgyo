# Pretendard 폰트 파일

이 디렉토리에 Pretendard 폰트 파일을 다운로드하여 배치하세요.

## 다운로드 방법

1. Pretendard 공식 GitHub에서 다운로드:
   - https://github.com/orioncactus/pretendard/releases
   - 또는 https://github.com/orioncactus/pretendard

2. 필요한 파일:
   - Pretendard-Regular.woff2
   - Pretendard-Medium.woff2
   - Pretendard-SemiBold.woff2
   - Pretendard-Bold.woff2

3. 또는 CDN 사용 (현재 설정):
   - CDN 링크가 이미 `app/layout.tsx`에 포함되어 있습니다.
   - 로컬 파일이 없어도 CDN을 통해 자동으로 로드됩니다.

## 참고

- CDN 방식이 더 간단하고 빠르게 적용 가능합니다.
- 로컬 파일을 사용하면 더 나은 성능과 오프라인 지원이 가능합니다.
